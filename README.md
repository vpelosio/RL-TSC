# Reinforcement Learning - Traffic Signal Control 

Progetto tesi di laurea triennale

Setup per sistemi con scheda video NVIDIA

```python3 -m venv .venv```

Per Linux: ```source .venv/bin/activate```

Per Windows: ```.venv\Scripts\activate.bat```

```pip install -r requirements.txt```

Per eseguire il training: ```python train.py```

Per eseguire i test: ```python test.py --id TEST_ID``` (--skip-stl permette di saltare i test dell'algoritmo smart traffic light)
